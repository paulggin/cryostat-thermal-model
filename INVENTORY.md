## Reproducibility

All modules run on stock Python 3.10+ with NumPy, SciPy, pandas, and Matplotlib. No proprietary fridge software, no IBM credentials, no GPU. Total runtime for the full pipeline (validation + sweep + plot regeneration) is under 30 seconds on a laptop.

`validate_and_sweep.py` writes the sweep results to `../outputs/data/sweep_T_MXC_vs_N_coax.csv` for downstream analysis when run inside the working project tree.
